count-type

Count Type

Names what is being counted by the <count> element, for example, images, contributors, algorithms, etc.

Remarks

This attribute may describe countable objects (for example, images, lemmas, or algorithms) for which there is no specifically named counting element. Alternatively, this attribute may be used with the <count> element set and a value such as “figure”, replacing a named counter such as <fig-count>.

Attribute Values

In Element

<count>
Value Meaning
The name of an object, element, structure The object being counted, for example, “contributor” when counting the number of authors or “image” for the number of images.
Restriction: This attribute must be specified if the element is used.
