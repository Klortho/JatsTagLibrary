journal-id

Journal Identifier of a Related Article

Identifier for the journal that contains the related article.

Usage

The @journal-id attribute can be used, along with the @vol, @issue, and @page attributes, to provide the metadata for the related article.

Attribute Values

In Element

<related-article>
Value Meaning
Text, numbers, or special characters A journal abbreviation or other journal identifier.
Restriction: This is an optional attribute; there is no default.
