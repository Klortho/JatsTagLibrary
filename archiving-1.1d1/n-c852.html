description

Description of a Private Character

Human-readable description of a defined private character (for example, “Arrow, normal weight, single line, two-headed, Northwest to Southeast”).

Remarks

The @name attribute for a defined private character <private-char> provides a Unicode-style name (an all uppercase name) for the private character, for example, “COMBINING CYRILLIC PALATALIZATION”. In contrast, the @description attribute for a defined private character <private-char> provides a rough visual description, for example, “combining vertical tilde”.

Attribute Values

In Element

<private-char>
Value Meaning
Text, numbers, or special characters A rough visual description of the appearance of the <private-char> character, for example, “Greek capital letter omega with tonos”.
Restriction: This is an optional attribute; there is no default.
