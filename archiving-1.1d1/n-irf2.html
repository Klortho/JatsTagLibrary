MathML 3.0 MathML Setup Module

External module in the base Suite that is named in the Module to Name the Modules (%modules.ent;) and then called from the this Tag Set to set up the namespaces and parameter entities for the MathML 3.0 elements

Remarks

This module then calls the MathML 3.0 modules to include the MathML elements in this Tag Set.
