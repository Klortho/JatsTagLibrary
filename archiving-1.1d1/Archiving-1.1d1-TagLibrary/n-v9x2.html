Accessibility Class Elements

Elements added to make it easier to process documents in ways that are more accessible to people with special needs (for example, the visually handicapped) and for the devices that assist them (for example, pronouncing software). The <alt-text> element holds a short phrase description of an object such as a graphic, and the <long-desc> element contains a more complete description of the content or intent of an object.
