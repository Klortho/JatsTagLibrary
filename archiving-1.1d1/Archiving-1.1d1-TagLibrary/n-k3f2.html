%just-rendition;

Just Rendition

Names an element mix containing only the simplest of the typographic emphasis elements, as well as those for subscript and superscript, for use in models that allow data characters plus this restricted mixture of elements.

Remarks

Implementorâ€™s Note: This parameter entity and the related parameter entity %rendition-plus; have been put in place to restrict the amount of variability a person can achieve when modifying the Tag Set through parameter entity redefinition in the DTD. Some elements have been set as #PCDATA plus one parameter entity, while others have been set to #PCDATA plus the other, in an effort to allow designers to modify entire groups of elements, but not change similar models individually. In this Archiving and Interchange Tag Set, this both of these parameter entities have been overridden to allow any phrase-level elements.
