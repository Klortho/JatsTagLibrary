alttext

Alternate Text (MathML Tag Set)

Use documentation for the Mathematical Markup Language (MathML) 2.0 Tag Set. (http://www.w3.org/TR/MathML2/)

Remarks

The @alttext attribute is used only within the mathematical expressions of MathML. The equivalent function, providing words that can be display “behind” a graphic or be pronounced over a graphic, is performed by the <alt-text> element for graphics (and other elements) within this Tag Set.

Attribute Values

In Element

<mml:math>
ValueMeaning
Text, numbers, or special charactersUse MathML documentation.
Restriction: This is an optional attribute; there is no default.
