link-type

Type of Link (for a Related Object)

How a <related-object> element is related to the current document (for example, to indicate a Table of Contents relationship).

Attribute Values

In Element

<related-object>
Value Meaning
Text, numbers, or special characters The type of relationship between the related object and the current document.
Restriction: This is an optional attribute; there is no default.
