%math.qname;

Name of the MathML Top-level Element

The name for the <mml:math> element in MathML equations. This is set up to as the base name “math” prefixed by the MathML prefix, which is set using %MATHML.pfx;.

Remarks

The MathML Table Tag Set provides a .qname so the name can be easily customized.

Content Model

<!ENTITY % math.qname   "%MATHML.pfx;math"                           >

Expanded Content Model

mml:math
