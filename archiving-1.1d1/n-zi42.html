%citation.class;

Citation Class Elements

Element class created to hold the <element-citation> and <mixed-citation> elements (which reference an external document, as used within, for example, the text of a paragraph) and the citation alternatives element <citation-alternatives> (which contains a single citation in multiple forms, for example, in two languages).

Content Model

<!ENTITY % citation.class
                        "citation-alternatives | element-citation | 
                         mixed-citation | nlm-citation"              >
