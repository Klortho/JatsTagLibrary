%colgroup.element;

Column Group Element Control For a Table

Holds the INCLUDE or IGNORE keyword to control a conditional marked section, that determines whether the <colgroup> element is defined.

Remarks

The NISO JATS table model parameter entity is used in the same way as the equivalent parameter entity in the XHTML 1.1 table model.

Content Model

<!ENTITY % colgroup.element  "INCLUDE" >
