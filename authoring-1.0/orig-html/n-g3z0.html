Common Tagging Practice

Since there are as many different uses of this Tag Set as there are users, “Best Practice” is an elusive concept, impossible to define for all users. But in many situations there are common practices, and it is sometimes possible to explain several good practices, allowing an archive or publisher to determine if one of them fits their situation. This section contains tagging write-ups, more extensive than is appropriate for any single element page, that offer such tagging advice and alternatives.
