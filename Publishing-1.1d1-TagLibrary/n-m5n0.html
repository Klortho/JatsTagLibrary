<sec-meta>

Section Metadata

Container element for section-level metadata that is different from the parent document.

Attributes

id Document Internal Identifier
xml:base Base

Content Model

<!ELEMENT  sec-meta     %sec-meta-model;                             >

Expanded Content Model

((contrib-group)*, (abstract)*, (kwd-group)*, permissions?)

Description

The following, in order:

  • <contrib-group> Contributor Group, zero or more
  • <abstract> Abstract, zero or more
  • <kwd-group> Keyword Group, zero or more
  • <permissions> Permissions, zero or one

This element may be contained in:

<app>, <bio>, <boxed-text>, <notes>, <sec>
