<journal-subtitle>

Journal Subtitle (Full)

Full subtitle of the journal in which the article was published.

Remarks

Historical Note: This element (<journal-subtitle>) was significantly remodeled from previous versions of this Tag Set. The current NISO JATS values are backwards compatible with the last NLM version, but not with earlier versions. Specifically, in prior versions the <journal-subtitle> element was allowed outside the (new) <journal-title-group> container element.

Text, numbers, or special characters, zero or more
