%address-link.class;

Address Link Class Elements

Linking elements that can be used inside addresses, for example, the three generic external links: <ext-link>, <email>, and <uri>.

Remarks

Historical Note: In earlier releases of this Tag Set, this parameter entity was named %address-elements; although, even in those releases, it functioned as a class.

Content Model

<!ENTITY % address-link.class
                        "email | ext-link | uri"                     >

Module

JATS-default-classes1.ent
