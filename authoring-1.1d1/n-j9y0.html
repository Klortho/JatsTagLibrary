Anonymous Content Model

Content model for the <anonymous> element

Remarks

Unlike the base Tag Suite, this Authoring Tag Set defines the <anonymous> element as an EMPTY element, i.e., an element that has no content. The element is merely a flag for processing systems. When present, <anonymous> can be used to generate the text “anonymous”, “anon.”, etc.
