product-type

Type of Product

Category of product described in the element (for example, “book”, “software”, “article” “issue”, “website”, “film”, “hardware”).

Attribute Values

In Element

<product>
Value Meaning
Text, numbers, or special characters The type of product being reviewed, for example, “website” or “videogame”.
Restriction: This is an optional attribute; there is no default.
