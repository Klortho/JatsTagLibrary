<publisher>

Publisher

Container element for information about the person, company, or other entity that published a work.

Attribute

content-type Type of Content

Content Model

<!ELEMENT  publisher    %publisher-model;                            >

Expanded Content Model

((publisher-name, publisher-loc?)+)

Description

Any one of:

  • The following sequence (in order), one or more times:
    • <publisher-name> Publisher’s Name
    • <publisher-loc> Publisher’s Location, zero or one

This element may be contained in:

<journal-meta>
