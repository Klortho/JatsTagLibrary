MathML Annotation Model

This Tag Set modifies the original MathML DTD definition of this entity from ANY to one or more <p> elements. The original definition is extremely broad and can be problematic in conversions.

Remarks

Since this parameter entity was named in the same way as the MathML Tag Set equivalent, it does not follow the naming conventions for this Tag Set: it is in mixed not lower case.

Content Model
