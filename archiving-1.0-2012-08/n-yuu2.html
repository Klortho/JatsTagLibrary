%block-display-noalt.class;

Display Class No Alternatives Elements

The same content as the block display class parameter entity (%block-display.class;) minus the <alternatives> element. This parameter entity is used for graphical or other image-related elements, whenever the use of the ordinary block display class parameter entity (%block-display.class;) would cause a duplication or where alternatives are not allowed.

Content Model

<!ENTITY % block-display-noalt.class
                        "address | array |
                         boxed-text | chem-struct-wrap |
                         fig | fig-group | graphic | media |
                         preformat | supplementary-material |
                         table-wrap | table-wrap-group"              >
