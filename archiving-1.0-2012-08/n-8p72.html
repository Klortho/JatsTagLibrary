%simple-phrase;

Simple Phrase-Level Textual Elements

Names elements that may be used almost anywhere text is used, for example, inside most kinds of titles

Remarks

In this Archiving and Interchange Tag Set, this parameter entity has been overridden to allow any phrase-level elements.
