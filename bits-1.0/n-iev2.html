sort-key

Sort Key for Index Entry

Provide an alternative version of the index entry for use as a sort key, for example, as ASCII version replacing an accepted character with an unaccented one.

Attribute Values

In Element

<index-entry>
Value Meaning
Text, numbers, or special characters. A version of the index entry for sorting
Restriction: This attribute may be specified if the element is used.
