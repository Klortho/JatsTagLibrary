xmlns:xi

XInclude Namespace Declaration

This is not an attribute, but the namespace pseudo-attribute. The value provides a namespace prefix to use for the XInclude elements. All namespace prefixes must be associated with a URL, and the prefix “xi” has been set to the URL for the World Wide Web Consortium (W3C) XInclude Recommendation.

Attribute Values

In Elements

<book>, <book-part>, <book-part-wrapper>, <xi:fallback>, <xi:include>
ValueMeaning
http://www.w3.org/2001/XIncludeThe prefix “xi” has been set to the URL for the W3C XInclude Recommendation.
Restriction: Restriction: This attribute value may not be changed.
