<nav-pointer-group>

Navigation Pointer Group

More than one <nav-pointer> element, in a group associated with a Table of Contents entry (<toc-entry>) or Index Entry (<index-entry>).

Attributes

id Document Internal Identifier
xml:base Base

Content Model

<!ELEMENT  nav-pointer-group  
                       (#PCDATA %nav-pointer-group-elements;)*       >

Expanded Content Model

(#PCDATA | nav-pointer)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <nav-pointer> Navigation Pointer

This element may be contained in:

<index-entry>, <toc-div>, <toc-entry>
