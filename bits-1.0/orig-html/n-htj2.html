%XLINK.xmlns.attrib;

XLink Namespace Declaration

Sets up the namespace prefix and URI to use in the XLink linking attributes. See W3C XLink documentation or use the W3C documentation for the XHTML1.1 table model. (http://www.w3.org/TR/xhtml-modularization/abstract_modules.html#s_tablemodule)

Remarks

The attribute-like “xmlns:xlink” is not a true attribute, but a namespace pseudo-attribute. All namespace prefixes must be associated with a URL, and the prefix “xlink” has been set to the URL for the World Wide Web Consortium (W3C) XLink Recommendation.

Content Model

<!ENTITY % XLINK.xmlns.attrib
     "xmlns:%;  CDATA           #FIXED '%;'"
>

Expanded Content Model

xmlns:xlink CDATA #FIXED 'http://www.w3.org/1999/xlink'
