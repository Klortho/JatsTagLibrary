from-where

From Where

Records the origin of a book part, for example, inside the narrative front matter, the body, the back matter, or supplementary material, when the book part is exchanged

Remarks

When a <book-part> is exchanged, the @from-where attribute cam be used to record the origin of the <book-part>, inside the narrative <front-matter>, the <body>, the rear matter (<back>,) or in supplementary material.

Attribute Values

In Element

<book-part-wrapper>
Value Meaning
back Back matter
body Body
front-matter Front matter
supplemental Supplementary material
Restriction: This is an optional attribute; there is no default.
