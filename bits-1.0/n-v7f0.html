<issn>

ISSN

International Standard Serial Number, the international code that uniquely identifies a serial publication title.

Attributes

content-type Type of Content
id Document Internal Identifier
publication-format Format of Publication
specific-use Specific Use
xml:base Base

Content Model

<!ELEMENT  issn         (#PCDATA %issn-elements;)*                   >

Expanded Content Model

(#PCDATA | x)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <x> X - Generated Text and Punctuation

This element may be contained in:

<book-meta>, <book-part-meta>, <collection-meta>, <element-citation>, <event>, <event-desc>, <mixed-citation>, <product>, <related-article>, <related-object>
