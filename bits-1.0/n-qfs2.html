xmlns:mml

MathML Namespace Declaration

This is not an attribute, but the namespace pseudo-attribute. The value provides a prefix to use for the MathML elements. All namespace prefixes must be associated with a URL, and the prefix “mml” has been set to the URL for the World Wide Web Consortium (W3C) MathML Tag Set.

Attribute Values

In Elements

<book>, <book-part>, <book-part-wrapper>, <dedication>, <foreword>, <front-matter-part>, <mml:math>, <preface>
ValueMeaning
http://www.w3.org/1998/Math/MathMLThe prefix “mml” has been set to the URL for the W3C MathML Tag Set.
Restriction: Restriction: This attribute value may not be changed.
