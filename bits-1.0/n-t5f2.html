%counts-model;

Counts Model

Content Model for the <counts> element, a container element holding all the metadata elements that are “counts” of other elements, for example, the number of figures, the number of tables, the number of pages, etc., in a document

Content Model

<!ENTITY % counts-model "(count*, fig-count?, table-count?, 
                          equation-count?, ref-count?, page-count?, 
                          word-count?)"                              >
