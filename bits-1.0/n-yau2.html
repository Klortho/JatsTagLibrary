encoding

XInclude Encoding

The encoding to use for the resource being included. The encoding attribute has no effect when parse="xml".(See the W3C XInlcude documentation http://www.w3.org/TR/xinclude/.)

Attribute Values

In Element

<xi:include>
Value Meaning
Text, numbers, or special characters. A valid encoding name.
Restriction: This attribute may be specified if the element is used.
