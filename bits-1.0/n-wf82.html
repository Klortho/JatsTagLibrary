answer-type

Type of Answer

Used to mark an answer or a set of answers as a particular type for selection or processing.

Attribute Values

In Elements

<answer>
<answer-set>
ValueMeaning
essayThis answer requires a longer written answer from the student.
fill-in-the-blankThis answer is a medium length written answer, typically a phrase or sentence.
multi-selectThere are answer choices presented, and the student must select one or more.
multiple-choiceThere are answer choices presented, and the student must select one.
short-answerThis answer is a short length written answer, typically a word or very short phrase or sentence.
true-falseThe student must choose between “true” and “false”.
Restriction: This attribute may be specified if the element is used.
