Indexed

Indicates whether a unit (e.g., a book, dedication, chapter, etc.) should be processed by indexing software or human indexers

Remarks

Note: Indexing is often turned off (value “no”) for bibliographies and other references, as opposed to the body of a document.

Attribute Values

In Elements

<book>, <book-part>, <dedication>, <foreword>, <front-matter-part>, <preface>
Value Meaning
no Indexing is turned off, i.e., indexing software or human indexers should not index this unit.
yes Indexing is turned on, i.e., indexing software or human indexers should index this unit.
Restriction: This is an optional attribute; there is no default.
