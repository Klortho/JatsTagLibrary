nav-pointer-type

Type of Navigation Pointer

Indicates the kind of navigation pointer (<nav-pointer>) being used in a structural index entry, i.e., the pointer could identify a single page (value “point”) in the narrative text, or the beginning and end of a page range (values “start-of-range” and “end-of-range” respectively).

Attribute Values

In Element

<nav-pointer>
Value Meaning
end-of-range The end of a page range
point A single page
start-of-range The start of a page range
Restriction: This is an optional attribute; there is no default.
