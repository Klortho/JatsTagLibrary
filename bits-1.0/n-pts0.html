<event-desc>

Event Description

This is a description of an event in the publication history of a book, for example, reprinting or publication of the revised online edition. This text field may be as simple a few words or as complex as a narrative with embedded markup.

Attributes

id Document Internal Identifier
specific-use Specific Use
xml:base Base
xml:lang Language

Content Model

<!ELEMENT  event-desc   (#PCDATA %event-desc-elements;)*             >

Expanded Content Model

(#PCDATA | book-id | edition | date | string-date | issn | issn-l | isbn | pub-date)*

Description

Any combination of:

  • Text, numbers, or special characters
  • <book-id> Book Identifier
  • <edition> Edition Statement, Cited
  • <date> Date
  • <string-date> Date as a String
  • <issn> ISSN
  • <issn-l> ISSN-L (Linking ISSN)
  • <isbn> ISBN
  • <pub-date> Publication Date

This element may be contained in:

<event>
