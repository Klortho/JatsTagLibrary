<collection-id>

Collection Identifier

An identifier, such as a DOI, archive control number, or LCCN for a series or set of books.

Remarks

The @collection-id-type attribute should be used to name the type of identifier, such as an “lccn”. There is a separate element for any ISBNs (<isbn>) related to a collection.

Attributes

collection-id-type Type of Collection Identifier
id Document Internal Identifier
specific-use Specific Use
xml:base Base
xml:lang Language

Content Model

<!ELEMENT  collection-id
                        (#PCDATA)                                    >

Description

Text, numbers, or special characters

This element may be contained in:

<collection-meta>
