%col.element;

Column Element Control For a Table

This parameter entity holds the INCLUDE or IGNORE keyword to control a conditional marked section, that determines whether the <col> element is defined.

Remarks

The NISO JATS table model parameter entity is used in the same way as the equivalent parameter entity in the XHTML 1.1 table model.

Content Model
