<award-group>

Award Group

Container element for information concerning one award under which the work (or the research on which the work was based) was supported.

Remarks

There may be more than one source of funding, a grant or contract number, principal investigator, etc.

This element replaces the obsolete <contract-num> element.

Attributes

award-type Type of Award
id Identifier
rid Reference to an Identifier
specific-use Specific Use
xlink:actuate Actuating the Link
xlink:href Href (Linking Mechanism)
xlink:role Role of the Link
xlink:show Showing the Link
xlink:title Title of the Link
xlink:type Type of Link
xmlns:xlink XLink Namespace Declaration
xml:lang Language

Content Model

<!ELEMENT  award-group  %award-group-model;                          >

Expanded Content Model

(funding-source*, award-id*, principal-award-recipient*, principal-investigator*)

Description

The following, in order:

  • <funding-source> Funding Source, zero or more
  • <award-id> Award Identifier, zero or more
  • <principal-award-recipient> Principal Award Recipient, zero or more
  • <principal-investigator> Principal Investigator, zero or more

This element may be contained in:

<funding-group>
