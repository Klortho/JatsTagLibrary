%display-back-matter.class;

Display Element Back Matter Class Elements

Various attribution and copyright elements that may occur at the end of tables (<table-wrap>), figures (<fig>), and other block-level elements when such elements need a separate copyright, license, or attribution, etc.

Remarks

Easily confused with the parameter entity %display-back-matter; that presents a structured sequence involving similar elements (<attrib> and <permissions>). This structured sequence is used at the end of boxed text, a figure, or similar display element within this Tag Set only. The OR group named in %display-back-matter.class; is used in all four of the Tag Sets.

Content Model

                        "attrib | permissions"                       >
