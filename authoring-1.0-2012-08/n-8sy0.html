Default Element Classes Module

External module in the base Suite that sets up the default parameter entities and element class definitions that are used to create the element mixes and define the content models for the base Suite

Remarks

It is expected that some or all of the parameter entities declared in this module will be overridden in the Tag-Set-specific custom classes module, %articleauthcustom-classes.ent;.
