%inside-cell;

Inside Table Cell Elements

Elements that can be mixed with the data characters inside a table cell (inside the elements <td> or <th> element in the NISO JATS table model (based on and designed to be converted easily to the XHTML 1.1 table model).

Remarks

This parameter entity should be used as the value of the table-model-specific parameter entities that name the content of table cells, for example, the parameter entity %FlowMix; for the NISO JATS XHTML-based table and the parameter entity %paracon; for the OASIS XML Exchange (CALS) table model.
