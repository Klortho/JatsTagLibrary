%simple-text;

Simple Textual Content

Names elements that may be used inside elements that are really expected to be data characters (#PCDATA) and not to contain any of these things

Remarks

This parameter entity is for modeling elements that an authoring Tag Set could restrict to merely data characters (#PCDATA). In the base Suite, this parameter entity names elements that may be used inside elements that are really expected to be data characters (#PCDATA) and not to contain any of these things. Therefore, in the base Suite, this parameter entity is even simpler than the %simple-phrase; parameter entity: there is no math, and there are no links. An archival and interchange Tag Set dare not be that restrictive, since it needs to be able to handle the less mainstream cases. In this Archiving and Interchange Tag Set, this element mix has been overridden to allow any phrase-level elements.
