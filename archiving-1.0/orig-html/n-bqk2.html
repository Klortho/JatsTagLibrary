currency

Currency

Type of currency in which the price is presented.

Remarks

Best Practice: Although this is an optional attribute, best practice would be to include it for every price where it is known. The attribute can take any value, but best practice is to use the ISO 4217 “currency code” values. These are 3-letter, all upper case codes that are NOT the same as the ISO country codes. Examples include:

USD

United States Dollar in the US and US territories

CAD

Canadian Dollar

EUR

European Union Euro

RUB

Russian Ruble
These codes are defined by ISO in ISO 4217 and listed at: http://www.iso.org/iso/en/prods-services/popstds/currencycodeslist.html#SZ.

Attribute Values

In Element

<price>
Value Meaning
Text, numbers, or special characters An abbreviation for a currency, typically using the ISO 4217 three-letter alphabetic codes, for example, “USD” for the United States Dollar.
Restriction: This is an optional attribute; there is no default.
