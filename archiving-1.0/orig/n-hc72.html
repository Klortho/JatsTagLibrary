%related-object-elements;

Related Object Elements

Elements that can be included along with data characters inside the content model of the <related-object> element.

Remarks

Design Note: The parameter entity %all-phrase; was replicated and not used directly inside this parameter entity because the %article-link.class; elements are repeated in %references.class; and would therefore cause duplication.
