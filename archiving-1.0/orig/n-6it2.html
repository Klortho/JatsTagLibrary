Article/Publication Identifier Types

The @pub-id-type attribute names the type of identifier, or the organization or system that defined an identifier, for an identifier of the document or an identifier of a cited publication or other referenced object. This parameter entity holds an explicit list of values for the @pub-id-type attribute.
