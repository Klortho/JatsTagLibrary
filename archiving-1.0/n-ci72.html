%simple-display-noalt.class;

Simple Display Elements

The same content as the simple display class parameter entity (%simple-display.class;) minus the <alternatives> element. Both parameter entities name the simplest and most basic of the display class elements, which may be allowed in many places, for example, inside other display class elements or inside the cell of a table. This parameter entity is use in preference whenever the use of the ordinary simple display class (%simple-display.class;) would cause a duplication of the element <alternatives> or where <alternatives>s are not allowed.

Content Model

<!ENTITY % simple-display-noalt.class
                        "array | graphic | media | preformat"        >
